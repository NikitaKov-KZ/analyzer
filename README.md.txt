# Анализатор рисков госзакупок

Интеллектуальное приложение на Python для оценки уровня риска тендеров и выявления аномалий в государственных закупках




# Функционал


Загрузка (или генерация) данных о закупках
Применение набора эвристик:
Высокая цена
Повторные выигрыши у одного поставщика
Дробление лотов
Отсутствие конкуренции
Подсчёт общего риска

Назначение категории:
green (без риска)
yellow (умеренный риск)
orange (высокий риск)
red (очень высокий риск)

Итоговая оценка риска:
Уровень Green Yellow Orange Red
Условие	  0 	 1 	  2-3    4 

Визуализация результатов:
Таблица с подсветкой
Гистограмма распределения рисков



# Интерфейс



Интерфейс реализован на Streamlit. При запуске:
1. Выводится кнопка "Запуск"
2. После анализа — таблица с закупками и их рисками
3. Ниже — диаграмма по всем уровням риска
